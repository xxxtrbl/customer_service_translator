Step welcome
	Speak  您好,请输入您的姓名
	Listen $name
	Branch True , mainProc
	Branch False , createProc
	Silence silenceProc
	Default defaultProc
Step createProc
	Speak 用户名不存在! 请输入您要创建的用户名
	Listen $name
	Default mainProc
Step mainProc
	Speak 您好 + $name + ,请输入您的请求 1.投诉 2.余额 3.充值 4.退出
	Listen 
	Branch 投诉 , complainProc
	Branch 余额 , billProc
	Branch 充值 , chargeProc
	Branch 退出 , thanks
	Silence silenceProc
	Default defaultProc
Step chargeProc
	Speak 请输入您的充值金额
	Listen $money
	Default defaultProc
Step complainProc
	Speak 您的意见是我们改进工作的动力,请问您对我们有什么意见?
	Listen
	Default thanks
Step thanks
	Speak 感谢您的来电,再见
	Exit
Step billProc
	Speak 您的余额是 + $amount + 元
	Default mainProc
Step silenceProc
	Speak 听不清,请您重新输入
	Listen
	Branch 投诉 , complainProc
	Branch 账单 , billProc
	Silence silenceProc
	Default defaultProc
Step defaultProc
	Speak 请您做出选择!否则自动退出! 1.主页 2.退出
	Listen
	Branch 主页 , mainProc
	Branch 退出 , thanks
	Silence thanks
	Default thanks
